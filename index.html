Survival Analysis

R allows the programmer to use time-to-event data to create a survival object. This object can be plotted into a Kaplan- Meier plot, and specifications can be given to make the plot more elaborate. Cox proportional hazard regression and accelerated failure-time models. It is easy to plot cumulative hazard as well as generate other plots to choose distributions and make decisions about modeling approaches
